# SWGEMU Damage Calculator

A web-based damage calculator for Star Wars Galaxies Emulator (SWGEMU). Calculate DPS (Damage Per Second) for various weapons, professions, and special abilities.

## Features

- Calculate DPS for multiple professions (Bounty Hunter, Carbineer, Commando, Fencer, Pikeman, Pistoleer, Rifleman, Swordsman, Teras Kasi Master, and Jedi)
- Preloaded weapons and enemies for quick calculations
- Support for PvP calculations with shields and mitigation
- Sortable results table
- Support for lightsaber calculations with Force cost tracking

## Usage

1. Select a profession from the dropdown
2. Choose a weapon (or use preloaded weapons)
3. Set weapon stats (speed, min/max damage, damage type, armor piercing)
4. Configure enemy resistances and armor
5. View calculated DPS in the sortable table
